今回はURLの構成要素であるプロトコル、サーバ、パスやWebページの部品であるHTML ,CSS,Javascript, 画像・音声・動画についての説明と、HTML文書の構造について説明を受けました。また、HTML文書の作成の中の ハイパーリンク、画像の埋め込みについて演習1-3.1-4で実験しました。以下のリンクは実際の演習です。

演習の成果

  • 演習1-3: HTML文書の作成
  • 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回はHTMLを装飾する技術であるCSSについて学習しました。CSSをHTMLに適用する方法と、CSSの書式、色付け、フォント、グラデーションなどCSSのさまざまなプロパティついて学び、練習問題2-3を通して実験しました。

演習の成果

  • 演習2-3: ウェブブラウザ説明ページの装飾

今回は、JavaScriptの基本的な部分について学習しました。また、JavaScriptによってできること、JavaとJavaScriptの違いについても学びました。前回学んだウェブ開発ツールの中のコンソールの部分と、デバックについても学びました。

  • 演習3-6: オブジェクトの参照
